2. CMD

  • dir - Listado completo de directorios (.archivo-oculto)
  • cd - change directory [cd nombre-directorio] <DIR> es un directorio
  • cd . - no te mueves del directorio
  • cd .. - te regrasas un directorio
  • cd / - te va llevar a la raiz del sistema de ficheros ( C:\>)
  • cd ~ - no la reconoce windows/en linux te lleva al directorio home del ususario
  • cd C:/xampp/htdocs - te lleva a localhost de xampp
  • cls - significa clean screen (limpia la pantalla)
  • shutdown -r - Nos va reiniciar el equipo
  • shutdown -s - Nos va apagar el equipo
  • shutdown -l - Nos va a cerrar la sesión
  • exit - Nos va a cerrar la consola
  • C:\Users\Andres>dir - Nos va a mostrar mis directorios
  • C:\Users\Andres>cd Desktop - Nos va a llevar a C:\Users\Andres\Desktop>
  • C:\Users\Andres\Desktop>mkdir carpetapruebas - nos va a crear 21/05/2025 07:25 p. m. <DIR> carpetapruebas
  • rmdir carpetapruebas - elimina carpetapruebas
  • ""hola"" > pruebas.txt - crea un archivo pruebas.txt
  • del pruebas.txt - elimina el archivo
  • copy pruebas2.txt archivos_copiados\copia_pruebas2.txt previamente creado el directorio C:\Users\Andres\Desktop\carpetap>mkdir archivos_copiados
  • move pruebas.txt archivos_copiados - nos mueve dentro de la carpeta el archivo pruebas.txt
  • dir archivos_copiados - nos lista el contenido de esa carpeta
  • pru + tab - nos autocompleta
  • ipconfig - muestra conexiones de red, direcciones ip, equipo local
  • ipconfig /all - muestra mas opciones de red
  • help - muestra todos los comandos de windows con una descripcion
  • help find - Busca una cadena de texto en uno o más archivos.
  • find "pru" *.txt ---------- PRUEBAS2.TXT
  • find "PrU" *.* ---------- PRUEBAS2.TXT
  • find "" *pruebas2* --------- PRUEBAS2.TXT
  • find "" *pr??bas2* ---------- PRUEBAS2.TXT
  • find "" *pr??bas2* > salida.txt - nos crea el archivo con el nombre del archivo a buscar
  • more salida.txt - nos muestra por consolo el contenido del txt
  • dir > test.txt - genera un archivo con el resultado del dir
  • time - La hora actual es: 22:04:21.98 - Escriba una nueva hora:
  • composer --version - Composer version 2.8.1 2024-10-04 11:31:01
  • git --version - git version 2.45.1.windows.1
  • tasklist - nos muestra todas las tareas que tenemos
  • taskkill /F /IM CalculatorApp.exe - mata proceso de calculadora
